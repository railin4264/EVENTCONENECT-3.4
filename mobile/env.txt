# ==============================
# 📋 INSTRUCCIONES PARA OBTENER APIs:
# ==============================

# 🗺️ GOOGLE MAPS API:
# 1. Ve a: https://console.cloud.google.com/
# 2. Crear nuevo proyecto o seleccionar existente
# 3. Habilitar APIs: Maps SDK, Places API, Geocoding API
# 4. Crear credencial > API Key
# 5. Configurar restricciones de aplicación

# 🔐 GOOGLE OAUTH:
# 1. Ve a: https://console.cloud.google.com/apis/credentials
# 2. Crear credenciales > ID de cliente OAuth 2.0
# 3. Tipo: Aplicación Android/iOS
# 4. Configurar nombre del paquete y SHA-1 fingerprint

# 📘 FACEBOOK LOGIN:
# 1. Ve a: https://developers.facebook.com/
# 2. Crear nueva aplicación
# 3. Configurar Facebook Login
# 4. Agregar plataforma Android/iOS
# 5. Configurar Bundle ID y Key Hashes

# 🔔 EXPO PUSH NOTIFICATIONS:
# 1. No necesita configuración adicional
# 2. Se genera automáticamente en tiempo de ejecución

# ☁️ CLOUDINARY (Imágenes):
# 1. Ve a: https://cloudinary.com/
# 2. Crear cuenta gratuita
# 3. Dashboard > Account Details
# 4. Copiar Cloud name, API Key, API Secret

# 📧 SENDGRID (Emails):
# 1. Ve a: https://sendgrid.com/
# 2. Crear cuenta
# 3. Settings > API Keys
# 4. Crear Full Access API Key

# 💾 MONGODB:
# 1. Local: mongodb://localhost:27017/eventconnect
# 2. Cloud: https://www.mongodb.com/cloud/atlas
# 3. Crear cluster gratuito
# 4. Copiar connection string

# 📊 SENTRY (Monitoreo de errores):
# 1. Ve a: https://sentry.io/
# 2. Crear proyecto React Native
# 3. Copiar DSN del proyecto
